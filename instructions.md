# Instructions

### Dependencies
- bcrypt
- cookie-parser
- dotenv
- ejs
- express
- jsonwebtoken
- mongoose
- nodemon
- validator

### Environment Variables
add the `config.env `file which contains the environment variables:
PORT= `port number`
MONGODB_URL= `mongodb cluster url`
SECRET_TOKEN= `your secret token value`

### Start the application

- Install dependencies
```bash
npm install
```
- Start the server
```bash
npm start
```
